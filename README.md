# Mini App — Sign Up / Sign In

## Tətbiq haqqında
Bu tətbiq istifadəçi qeydiyyatı və giriş sistemidir.
- **UI**: React
- **API**: ASP.NET Core
- **Database**: MSSQL

## İşə salma

### Tələblər
- Docker Desktop quraşdırılmış olmalıdır

### Addımlar

1. Layihəni yüklə:
git clone <repo-link>
cd mini-app


2. Tətbiqi işə sal:
docker compose up -d --build


3. Brauzerdə aç:
- UI: http://localhost:3000
- API: http://localhost:5000

## İstifadə
1. http://localhost:3000 ünvanını aç
2. "Qeydiyyat" səhifəsindən hesab yarat
3. "Daxil ol" səhifəsindən giriş et
4. Uğurlu girişdən sonra "Xoş gəldin!" səhifəsi açılır
